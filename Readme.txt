
GPUProfiler

■ 用途

- Unity 上で、Glaxy S6/Galaxy S6 edge の GPU 情報を出力するためのコンポーネントです。

■ 導入方法

- 任意のゲームオブジェクトに GPUProfiler.cs をアタッチしてください。
